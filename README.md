# Cha_E

[CHAI] 3D Virtual Solution

: 단 1%의 '차이'가 만드는 압도적인 퀄리티

안녕하세요, **테크니컬 3D 아티스트 '차이(CHAI)'**입니다.

단순히 예쁜 아바타를 만드는 것을 넘어, 당신의 방송 환경과 캐릭터의 성격까지 설계하는 올인원 솔루션을 제공합니다.

남들과 똑같은 양산형 아바타가 아닌, **기술력과 디테일의 '차이'**를 경험해 보세요.

## 1. Visual Difference [깊이가 다른 비주얼]

"어떤 각도, 어떤 조명에서도 당신은 빛나야 합니다."

기존 에셋을 그대로 사용하는 것이 아니라, '차이'만의 노하우로 재해석하여 세상에 하나뿐인 룩(Look)을 완성합니다.

- **Custom Texture & Retouching**: Substance Painter와 Photoshop을 활용해 피부 질감, 메이크업, 헤어 브릿지 등을 직접 드로잉하여 독보적인 분위기를 연출합니다.

- **Clean Topology**: 3D 모델의 기초인 토폴로지(폴리곤 흐름)를 정교하게 다듬어, 조명을 받을 때 생기는 렌더링 그림자 오류를 원천 차단합니다.

- **Perfect Calibration**: 텍스처 색보정, 매트캡(Matcap) 하이라이트 수정, 실시간 그림자(Real-time Shadow) 설정을 통해 유니티와 방송 화면 간의 괴리감을 없앱니다.

## 2. Expression Difference [영혼이 담긴 페이셜]

"캐릭터의 성격(Personality), 표정 하나하나에 담아드립니다."

단순히 움직이는 얼굴이 아닙니다. 당신의 방송 스타일과 목소리 톤에 딱 맞는 '살아있는 표정'을 설계합니다.

- **Personality Facial**: 의뢰분의 성격(활발함/소심함/도도함 등)을 분석하여 웃는 모양, 눈썹의 각도 등을 커스터마이징합니다.

- **Dynamic Mouth Movement**: 답답하게 웅얼거리는 입모양은 그만. 시원하게 찢어지는 입매와 고가동역 구강 구조를 구현하여 말하는 맛을 살려드립니다.

- **Unbreakable Rigging**: 과격한 표정을 지어도 그래픽이 깨지거나 뭉개지지 않도록 극한의 표정 테스트를 거쳐 완성합니다.

- **Signature Expressions**: 홍조, 멘붕, 청색증 등 방송의 재미를 더할 특수 표정과 기믹을 제작합니다.

## 3. Technical Difference [방송 최적화 기술력]

"방송 사고 없는 쾌적함, 최신 기술의 집약체."

유니티(Unity) 스크립트와 코드를 직접 다루는 엔지니어링 역량으로, 가장 안정적이고 효율적인 아바타를 제공합니다.

- **Warudo Specialist (와루도 특화)**: 와루도 최신 기능(API, 신기능) 완벽 대응 및 업데이트 지원
  - Node/Blueprint 설계를 통한 역동적인 상호작용(PU 설정 등) 구현
  - 방송용 소품(Props) 제작 및 제어 시스템 세팅

- **Advanced Physics**: Magica Cloth와 Spring Bone을 복합 사용하여 머리카락과 의상의 찰랑임을 자연스럽게 구현하며, 손이 옷을 뚫지 않는 완벽한 콜라이더(Collider) 값을 찾아냅니다.

- **Optimization**: 자체 에셋과 스크립트를 활용하여 불필요한 연산을 줄이고, 작업 속도는 높이면서 퀄리티는 유지합니다.
